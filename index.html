Sobre o ePulpito

O ePúlpito, ideia concedida pelo professor Alberto Ciarcia Júnior, é uma solução inovadora para apresentações interativas em auditórios e eventos. Construído em madeira MDF e equipado com uma tela touchscreen, o sistema opera com o BigLinux e hardware específico, permitindo que o palestrante controle a apresentação sem depender de um computador, o que reduz interrupções. Estudos indicam que tecnologias interativas, como o ePúlpito, melhoram a retenção de informações. O sistema tem sido bem avaliado por sua eficácia e praticidade, com um script automático para manter o software sempre atualizado. O projeto contribui para o auditório ao promover tecnologias acessíveis e de código aberto e serve como um modelo para futuras inovações em apresentações.

Nosso Projeto

Hardware

O hardware consiste em uma doação do professor Alessandro Garcia e se consiste em:

  • I3 de 2° Geração
  • 4GB de memória RAM
  • SSD de 120 GB

Material Usado - Madeira MDF

O MDF é uma excelente escolha para montar um púlpito porque oferece um acabamento suave e de alta qualidade, é mais estável e durável do que o MDP e o plástico, e é mais fácil de trabalhar do que o metal. Além disso, proporciona um bom custo-benefício.

Sistema Operacional - BigLinux

O BigLinux é uma distribuição brasileira do sistema operacional Linux, baseada no Debian. É projetado para ser fácil de usar, com uma interface amigável e aplicativos pré-instalados, ideal para quem quer uma alternativa simples e funcional ao Windows ou macOS.
